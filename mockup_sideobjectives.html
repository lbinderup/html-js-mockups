Formalized System Rules

  • Base Structure: The core timeline is a strict repeating pattern of 3 Main Objectives followed by 1 Boss Objective.
  • Boss Exclusivity: Bosses never have side objectives. When calculating spacing for side objectives, Bosses are completely ignored (they essentially "pause" the logic).
  • Initial Start: The very first side objective has an equal chance to appear on the 1st, 2nd, or 3rd Main objective.
  • Cooldown (Min Gap): Once you get a side objective, the next Main objective will never have one. This guarantees a minimum gap of 1 Main objective.
  • Pity Timer (Max Gap): You cannot go 3 Main objectives without seeing a side objective. If you have skipped 2 Main objectives, the 3rd one is guaranteed to have a side objective.
  • Anti-Streak Variance: To prevent the game from feeling like a predictable "every other" metronome, the system actively weights the RNG to alternate between skipping 1 Main and skipping 2 Mains.
